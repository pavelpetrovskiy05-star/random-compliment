# Random Compliment Generator
Петровский Павел группа миб-301

## Описание
Программа выводит случайный комплимент.  
Фичи:
- Персонализированное приветствие с именем пользователя  
- Выбор категории комплиментов: внешность, характер, таланты  


## Быстрый старт
1. Клонируем репозиторий:
git clone <URL вашего репозитория>
2. Переходим в папку:
cd random-compliment
3. Запускаем:
python main.py
4. Вводим имя и выбираем категорию комплимента
